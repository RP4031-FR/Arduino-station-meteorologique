# Arduino-station-meteorologique
Dans ce projet , vous aller configurer une station météorologique pour mesurer la température et l'humidité , et afficher les valeurs sur un écran LCD
Attention : Il peut avoir des problèmes de codes !!!

Le code que j’ai utilisé est directement recopié du livre :Les meilleurs projets ARDUINO. Je ne l’ai pas modifié ni adapté : il s’agit exactement du programme tel qu’il est présenté dans l’ouvrage.
Je l’ai fait parce que je ne voulais pas vous faire payer les 14,90 € demandés pour obtenir le contenu, et ducoup j'ai donc simplement repris le code tel qu’il apparaît dans le livre.
